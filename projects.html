Project 3: Insurance Policy Management Web Application

This project was a solo endeavor where I designed and developed an Insurance Policy Management Web Application using Java, JSP, and XML. The application featured operational hyperlinks, a persistent database, and robust CRUD functionality. It involved a range of technologies, including Java Persistence API, Model View Controller architecture, JavaBeans, Java Server Pages, Spring Security, and MySQL. This project exemplified my ability to work independently and manage the entire software development lifecycle.

Technical Specifications

  • Programming Languages: Java, JSP, and XML
  • Database Management: Utilized mySQL for creating and managing a persistent database.
  • Technical Components: Implemented Java Persistence API, Model View Controller (MVC) architecture, JavaBeans, Java Server Pages (JSP), and Spring Security.
  • Functionality: Featured operational hyperlinks, robust CRUD (Create, Read, Update, Delete) commands, and secure user authentication.
  • Independent Development: Solely responsible for designing, coding, and managing the complete software development lifecycle.

Project 4: FitQuest, a Diet and Weight Loss Web Application

This standout project, developed within a diverse team of five, led to the creation of FitQuest, a Diet and Weight Loss Web Application. It involved problem statement formulation, user story development, and agile project management techniques. We presented the application at the Monash University IT Postgraduate Expo and received feedback from industry mentors. This experience not only showcased my technical skills but also highlighted my ability to collaborate effectively and implement user-centric design principles, adhering to agile methodologies. I am extremely proud of this project, which was the Monash University Industrial Experience Postgraduate Expo winner for the topic of Health.

Technical Specifications

  • Team Collaboration: Developed within a multi-disciplinary team of five members.
  • Project Management: Utilized agile methodologies, including user story creation, stand-up meetings, and peer review sessions.
